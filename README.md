> A premium, open-source clipboard manager for your Mac menu bar.

## 📦 Installation

1. Download the latest `.dmg` from the [Releases page](https://github.com/Charlesprakasam/clipvault/releases/latest).
2. Open the `.dmg` file.
3. Drag **ClipVault.app** into your **Applications** folder.

### Bypassing macOS Security (First Launch Only)

Because the app is open-source and not signed with a paid Apple Developer certificate, macOS will block it on the first launch. To bypass this:

**Method 1: The Right-Click Trick (Fastest)**
1. Open your **Applications** folder in Finder.
2. Find `ClipVault.app`.
3. Right-click (or Control-click) the app icon and select **Open**.
4. A warning dialog will still pop up, but this time it will have an **Open** button. Click it. *(macOS will remember it's safe for future launches).*

**Method 2: System Settings**
1. Launch ClipVault normally (it will show a warning).
2. Open **System Settings** → **Privacy & Security**.
3. Scroll down — you'll see "ClipVault was blocked".
4. Click **Open Anyway**.

After bypassing the warning, grant Accessibility permissions if prompted. ClipVault will appear as a 📋 clipboard icon in your menu bar.
